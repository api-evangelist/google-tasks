# Google Tasks API

The Google Tasks API lets you search, read, and update Google Tasks content and metadata. You can create, update, delete, and organize tasks across multiple task lists, move tasks between positions, and manage task completion status programmatically.

## Artifacts

- **OpenAPI**: [openapi/tasks.yml](openapi/tasks.yml) - OpenAPI 3.1.0 specification for the Google Tasks API.
- **JSON Schema**: [json-schema/tasks.json](json-schema/tasks.json) - JSON Schema (draft 2020-12) for Google Task resources.
- **JSON-LD**: [json-ld/tasks.jsonld](json-ld/tasks.jsonld) - JSON-LD context mapping Google Tasks resources to Schema.org vocabulary.
